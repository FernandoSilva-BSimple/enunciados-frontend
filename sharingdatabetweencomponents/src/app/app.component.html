Examples

@Input & @Output
  • This example uses the `app-collaborators-list` and `app-collaborator-details` components together.
  • `app-component` communicates with them through @Input and @Output.
  • The `app-collaborators-list` component displays a list of collaborators, and when a collaborator is selected, it emits an event that is handled by the parent component to update the `selectedCollaborator` property. The `app-collaborator-details` component then displays the details of the selected collaborator.
  • Because the `collaborator` property is passed to the `app-collaborator-details` component, it can be used to populate the form. In such case, if the value of `name` property is updated in the form, its value is reflected automatically in the list.
